# 📱 프로젝트 아이디에이션: 도파민 밸런스 서비스

## 📌 프로젝트 개요
현대 청년층은 짧고 강한 자극의 미디어(SNS·숏폼 영상 등)에 쉽게 노출되어 **도파민 중독**에 빠지고 있습니다.  
이로 인해 집중력 저하, 생산성 감소, 습관적 회피 행동 등의 부정적 영향을 겪습니다.  

**팀 Hibee**는 도파민 소비 패턴을 정밀하게 분석하고,  
**대체 행동 추천 + 보상 시스템 + 소셜 기능**을 결합하여 **지속 가능한 행동 변화**를 유도하는 서비스를 개발하고자 합니다.  
→ 단순한 차단이 아닌, **개인화·자율성 강화**를 통해 건강한 디지털 라이프스타일 정착을 목표로 합니다.  

---

## 🎯 핵심 키워드
- **도파민 중독 관리**
- **사용자 행동 데이터 분석**
- **맞춤형 행동 변화 유도**
- **힐링형 인터페이스**

---

## 👤 고객 페르소나
**Persona 1. 대학생 지민 (23세, 컴퓨터공학과)**  
- 시험 기간에도 SNS·숏폼을 반복적으로 시청 → 학업 집중 어려움  
- 모바일 앱·웨어러블 기기 사용에 익숙  
- **선호 동기부여:** 친구와 랭킹 경쟁, 캐릭터 꾸미기  

**Persona 2. 신입 직장인 민호 (26세, 마케팅 직군)**  
- 스트레스 후 야식·넷플릭스 몰아보기 → 수면 패턴 악화  
- 건강 관리 앱 경험 多, 데이터 기반 피드백 선호  
- **선호 동기부여:** 시각화된 리포트, 변화 수치 확인  

---

## 😣 Pain Point & 요구사항
| Pain Point | 요구사항 |
|------------|-----------|
| SNS·게임·간식 등 즉각적 보상 중독 | 강압적 차단 X → 재미있는 보상 시스템 필요 |
| 내가 언제/왜 폰을 보는지 모름 | **AI 리포트**: 시간·장소·감정별 사용 데이터 시각화 |
| 혼자 습관 유지 어려움 | 그룹 챌린지, 랭킹 시스템 제공 |
| 기존 앱은 지루함 | 캐릭터·아이템·뱃지 등 성장형 UX 필요 |
| 불규칙 수면·스트레스 | 웨어러블 데이터 기반 맞춤 미션 제공 |

## 🛠️ 주요 기능
1. **개인화 리포트**
   - 스크린타임 + 앱 사용 + 감정 입력 분석
   - 시간·상황·감정별 도파민 패턴 리포트 제공
   - “아침 SNS 사용 20% 감소” 등 변화 시각화

2. **대체 행동 추천**
   - 관심사/취미 기반 → 독서·명상·산책 등 추천
   - 성향 테스트·랜덤 탐색 기능
   - 사용자 만족도 평가 기반 추천 알고리즘 개선

3. **보상 시스템**
   - 개인 챌린지: 캐릭터 꾸미기, 아이템
   - 그룹 챌린지: 공동 목표, 뱃지, 랭킹 시스템
   - 실질적 리워드 연계 가능

4. **소셜 기능**
   - 챌린지 기록 공유
   - 그룹 경쟁·협력 통한 지속 동기 부여

5. **확장 기능**
   - 웨어러블·헬스 데이터 연동 (수면, 심박수 기반 미션)
   - 캘린더 기록 기능 (대체 활동 기록화)

---

## 🧩 기술 스택
- **Frontend:** React Native  
- **Backend:** FastAPI, PostgreSQL  
- **AI/ML:** 사용자 패턴 분석, 추천 알고리즘 (LSTM/Transformer, Hybrid CF)  
- **Wearable API:** Apple HealthKit, Google Fit  

---

## 🤖 AI 모델 설계 방향
- **도파민 소비 패턴 분석 모델**
  - 입력: 시간대, 앱 사용량, 감정 태그
  - 출력: 도파민 위험 시간대 예측
  - 모델: 시계열 기반 ML (LSTM/Transformer)

- **대체 행동 추천 모델**
  - 입력: 관심사, 성향, 과거 만족도
  - 출력: 개인 맞춤 추천 리스트
  - 모델: 협업 필터링 + 콘텐츠 기반 추천

- **챌린지 설계 모델**
  - Rule-based 시작 → 강화학습 기반 확장
